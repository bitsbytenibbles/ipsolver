The Dynamic Lot Sizing Problems module generates the quantities to be ordered over a known and fixed planning horizon for as single item with deterministic and discrete time varying demand. The problem solved by this module is often called the Dynamic Lot Sizing Problem (DLSP). The module can solve both cases when shortages are allowed or disallowed depending on the value entered for the backorder cost. Seven lot sizing procedures are implemented in this module. They are

  • Silver-Meal heuristic procedure.
  • Least Unit Cost heuristic procedure.
  • Part Period Balancing heuristic procedure.
  • Economic Order Quantity heuristic procedure.
  • Periodic Order Quantity heuristic procedure.
  • Lot-for-Lot heuristic procedure.
