The Joint Replenishment Problems module solves the problem of coordinating the replenishments of several items from a single supplier. Each time an order is placed, a major ordering cost is incurred, regardless of the number of items ordered. Moreover, an item-dependent minor ordering cost is incurred whenever the item is included in a replenishment order. The module provides solution procedures which assume that the major ordering cost is charged every basic cycle time and that the items are ordered every some integer multiple of this basic cycle. The solution procedures included in this module are:

  • Goyal (Management Science, 1973)
  • Hariga (Journal of Operational Research Society, 1994)
  • Jackson, Mucsktadt and Maxwell (IIE Transactions, 1985)
  • Silver (Management Science, 1976)
