The Resources Constrained Problems module addresses multi-items inventory lot sizing problems with limited resources. This module determines the ordering quantities that minimize the total inventory system cost without exceeding the resources capacities. It solves several multi-item inventory models under single and two restrictions with and without shortages. The types of constraints that are included in this module are:

  • inventory investment (budget) constraint
  • turnover ratio constraint
  • storage capacity constraint
  • number of orders/setups
  • setup capacity constraint
  • storage capacity and budget constraints
  • storage capacity and turnover ratio constraints
